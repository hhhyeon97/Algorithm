### 23881번 - 알고리즘 수업 - 선택 정렬 1 풀이 과정

ex ) 배열 [3, 1, 2, 5, 4]와 K=2일 때, 선택 정렬의 교환 과정

### 1. `초기 배열 : [3, 1, 2, 5, 4]`

- last = 4 (배열의 마지막 인덱스)

### 2. `1차 반복 (last = 4)`:

- A[0..4]에서 가장 큰 값 찾기 : `[3, 1, 2, 5, 4]` → 최대값은 `5` (인덱스 3)
- A[4]와 A[3] 교환: `[3, 1, 2, 4, 5]`
- 교환 횟수 증가 : `swapCount = 1`

### 3. `2차 반복(last = 3)`:

- A[0..3]에서 가장 큰 값 찾기 : `[3, 1, 2, 4]` → 최대값은 `4` (인덱스 3)
- A[3]와 A[3]은 교환하지 않음 (값이 동일)
- 교환 횟수 : 변동 없음 (여전히 `swapCount = 1`)

### 4. `3차 반복(last = 2)`:

- A[0..2]에서 가장 큰 값 찾기 : `[3, 1, 2]` → 최대값은 `3` (인덱스 0)
- A[2]와 A[0] 교환 : `[2, 1, 3, 4, 5]`
- 교환 횟수 증가 : `swapCount = 2`
- 2번째 교환이므로 결과 저장 : `swapResult = [2, 3]`
- 반복 종료 (swapCount가 K와 같기 때문에)

### 5. 최종 결과 출력 :

- swapCount는 K와 같으므로, `[2, 3]`을 출력

### 6. 결론

이 예시에서 배열 [3, 1, 2, 5, 4]의 2번째 교환은 2와 3이다. <br/>교환 횟수를 세면
서 K 번째 교환을 추적하면 결과를 정확히 찾을 수 있음 !
